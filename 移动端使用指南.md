# 移动端使用指南

## 方式1：在同一局域网内访问（推荐）

### 步骤：

1. **确保电脑和手机/平板连接同一WiFi**

2. **启动后端服务器**
   ```bash
   npm start
   ```
   服务器会显示运行地址，通常是 `http://localhost:3000`

3. **查找电脑的IP地址**
   - Windows: 打开命令提示符，输入 `ipconfig`，查找"IPv4 地址"
   - Mac/Linux: 打开终端，输入 `ifconfig` 或 `ip addr`，查找局域网IP

4. **在手机/平板上访问**
   - 打开浏览器（Safari、Chrome等）
   - 输入：`http://[电脑IP地址]:3000`
   - 例如：`http://192.168.1.100:3000`

### 修改服务器配置（如果需要）

如果需要从其他设备访问，需要修改 `server.js`：

```javascript
// 将这一行：
app.listen(PORT, () => {
// 改为：
app.listen(PORT, '0.0.0.0', () => {
```

这样服务器会监听所有网络接口，允许局域网访问。

## 方式2：使用手机热点

1. **开启手机热点**
2. **电脑连接手机热点**
3. **手机使用自己的热点网络**
4. **按照方式1的步骤操作**

## 方式3：部署到云服务器

### 使用免费托管服务

#### Vercel（推荐）

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **部署**
   ```bash
   vercel
   ```

3. **配置环境变量**（如果需要）

#### Netlify

1. 访问 https://www.netlify.com/
2. 注册账号
3. 拖拽项目文件夹到 Netlify
4. 自动部署完成

#### GitHub Pages（仅前端）

1. 将项目推送到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源

## 方式4：使用本地服务器工具

### 使用 VS Code Live Server

1. **安装 Live Server 扩展**
2. **右键点击 `index.html`**
3. **选择 "Open with Live Server"**
4. **在手机上访问显示的地址**

### 使用 Python 简单服务器

```bash
# Python 3
python -m http.server 8000

# 然后在手机访问：http://[电脑IP]:8000
```

## 常见问题

### Q: 手机无法访问
**A:** 检查：
- 电脑和手机是否在同一WiFi
- 防火墙是否阻止了端口3000
- 服务器是否正在运行

### Q: 图片加载失败
**A:** 确保：
- 图片路径正确
- 使用相对路径（如 `Picture/1.jpg`）
- 如果使用绝对路径，需要确保路径可访问

### Q: 触摸不灵敏
**A:** 已优化触摸区域，如果仍有问题，可以调整按钮大小

## 性能优化建议

1. **压缩图片** - 减小图片文件大小，加快加载速度
2. **使用CDN** - 如果部署到云服务，使用CDN加速
3. **启用缓存** - 浏览器缓存静态资源
